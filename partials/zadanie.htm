Задание

Направете уеб сайт, който да съдържа следните елементи:

  • 6 уеб страници
  • 1 подменю
  • 2 джаджи (widgets)
  • 1 вградено медийно съдържание

Изпълнение

  • джаджа 1: имплементирах бутон за сваляне на съдържанието в pdf формат като използвах библиотеката htmltopdf: 
    
          var cloned = document.getElementById('to-pdf').cloneNode(true)
          cloned.style.margin = '20px'
          html2pdf(cloned)
  • джаджа 2: имплементирах стилизиране на активната страница HTML CSS JAVASCRIPT като извлякох от url-а името на файла и след това добавих стилизиране на съответния елемент 
    
        var filename = new URL(window.location.href).pathname.split('/').pop()
        var withoutExt = filename.split('.')[0].toUpperCase()
        
        var collection = document.querySelectorAll('.w3-tag')
        collection.forEach(el => {
          if(el.textContent === withoutExt){
            el.classList.add("w3-black")
            el.classList.remove("w3-small")
            el.classList.remove("w3-grey")
          }
        })
